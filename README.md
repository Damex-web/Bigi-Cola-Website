# 🥤 Bigi Cola Website Clone

A modern and responsive **Bigi Cola landing page** built from scratch using **HTML, CSS, and JavaScript**.  
This project focuses on clean UI design, smooth interactions, and mobile responsiveness.

---

## 🚀 Features

- 🔥 Modern UI/UX design  
- 📱 Fully responsive layout (mobile-friendly)  
- 🎨 Smooth animations and transitions  
- 🍔 Mobile navigation menu (toggle + slide-in)  
- 🧊 Glassmorphism / blur effects  
- ⚡ Clean and structured code  




## 🛠️ Technologies Used

- **HTML5** – Structure  
- **CSS3** – Styling & animations  
- **JavaScript** – Interactivity  

---

## 🎯 What I Learned

- Building a real-world landing page from scratch  
- Creating responsive layouts for different screen sizes  
- Implementing mobile navigation with JavaScript  
- Designing modern UI with gradients and blur effects
